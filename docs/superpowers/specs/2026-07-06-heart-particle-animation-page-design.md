# 尹湘清 · 粒子爱心网页 — 设计文档

**日期：** 2026-07-06

## 目标

制作一个浪漫的单页网页：屏幕中央有一个由粒子组成的爱心，粒子持续飘动闪烁，颜色为蓝紫星空渐变（深蓝→紫色→粉红），背景为午夜蓝渐变，爱心正中央以白色优雅字体显示"尹湘清"。

## 技术方案

- 单文件 HTML（内嵌 CSS + JS），使用 Canvas 2D 绘制粒子系统
- 心形参数方程生成粒子位置分布
- 粒子持续微动 + 随机闪烁模拟萤火虫效果
- 文字使用系统手写体或引入 Google Fonts 手写字体

## 功能清单

| 需求 | 状态 |
|------|------|
| 午夜蓝渐变背景 | ✅ |
| 心形粒子汇聚动画 | ✅ |
| 蓝紫星空渐变粒子颜色 | ✅ |
| 粒子持续飘动闪烁 | ✅ |
| "尹湘清"居中白色手写体 | ✅ |
| 全屏响应式自适应 | ✅ |
