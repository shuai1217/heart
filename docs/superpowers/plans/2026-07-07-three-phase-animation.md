# 三阶段动画实现计划

**目标：** 将现有 3D 粒子爱心改造为三段式叙事动画，总时长约 30-40 秒，配合音乐高潮。

---

## 阶段一：星光汇聚（约 5-8 秒）

**视觉效果：**
- 屏幕初始为白色背景，布满随机散布的微小星光粒子（类似星空）
- 星光粒子缓慢向中心汇聚，逐渐形成 3D 爱心形状
- "尹湘清"三个字从透明到显现，带有打字机效果（逐字出现）

**技术实现：**
- 复用现有 HeartParticle 和 InnerParticle，但初始位置设为随机散布
- 每帧将粒子位置向 targetX/Y/Z 方向插值移动（lerp），形成汇聚效果
- 汇聚进度用 `easeOutCubic` 缓动，约 5 秒完成
- 文字逐字出现：3 个字，每个间隔 0.5 秒

**关键改动：**
- `heart.js` 中新增 `phase` 状态机
- 粒子初始位置改为随机散布，通过 `convergenceProgress` (0→1) 控制汇聚程度
- 文字显示改为逐字控制

---

## 阶段二：自转跳动（约 20-25 秒）

**视觉效果：**
- 爱心完整出现后，开始缓慢 3D 自转
- 粒子颜色从蓝紫渐变逐渐过渡为玫瑰红色系
- "尹湘清"三个字开始跳动（每个字的 scale 独立跳动，模拟心跳节奏）

**技术实现：**
- 颜色过渡：使用 `colorLerp` 函数，在 8-10 秒内将 colorStops 从蓝紫渐变逐步替换为玫瑰红
  - 外层粒子颜色范围：深粉 → 玫红 → 玫瑰红 → 浅粉
  - 内层粒子颜色范围：暖粉 → 玫瑰红 → 深玫红
- 文字跳动：每个字独立应用 CSS `transform: scale()` 脉冲，频率与心跳同步（lub-dub 双脉冲）
- 自转速度保持当前的 0.00005

**关键改动：**
- `heart.js` 中添加 `colorTransitionProgress` (0→1)
- `index.html` 中文字拆分为三个独立的 `<span>`，每个字独立跳动
- 心跳脉冲同步文字跳动

---

## 阶段三：花瓣爆炸（约 5-7 秒）

**视觉效果：**
- 爱心突然向外爆炸，粒子变为玫瑰花瓣形状
- 花瓣在屏幕上飘落、旋转、渐隐
- "尹湘清"三个字无缝变为"男孩和女孩拥抱"的画面

**技术实现：**
- 爆炸触发：将粒子目标位置设为随机向外飞散的方向，速度逐渐衰减
- 粒子形状从圆形改为花瓣形状（Canvas 绘制椭圆+尖端）
- 花瓣飘落：重力 + 风力 + 旋转，类似真实的落叶效果
- 文字消失后，在爱心中心绘制两个小人剪影（简笔画风格），做出拥抱姿态
- 小人随风微微摇摆，花瓣继续飘落

**关键改动：**
- `heart.js` 中添加 `RosePetal` 粒子类，替代圆形粒子
- 爆炸后 2-3 秒，在画布中央绘制拥抱小人剪影
- 音效切换：爆炸瞬间播放一个轻柔的"唰"声（用 Web Audio API 合成）
