# 3D 粒子爱心实现计划

**Goal:** 将现有 2D 粒子爱心升级为 3D 粒子爱心，支持缓慢自转 + 鼠标拖拽旋转 + 心跳 3D 缩放脉冲。

**Architecture:** 用数学公式生成 3D 爱心表面的粒子坐标，通过透视投影映射到 2D Canvas。保留现有的粒子闪烁、漂移、散射效果，增加 Z 轴旋转和深度排序绘制。

**Tech Stack:** 纯 Canvas 2D，无外部依赖

## Task 1: 3D 爱心数学模型 + 粒子生成

**文件:** `heart.js`

将现有的 2D 心形方程替换为 3D 心形参数方程：

```
x = 16 * sin³(t) * sin(u)
y = -(13 * cos(t) - 5 * cos(2t) - 2 * cos(3t) - cos(4t)) * sin²(u)
z = 12 * cos(u) + 3 * cos(2u) + 2 * cos(3u)
```

其中 `t ∈ [0, 2π]`, `u ∈ [0, π]`。

改动要点：
- 每个粒子存储 `targetX, targetY, targetZ`（3D 坐标）
- 外层粒子（HeartParticle）分布在 3D 爱心表面
- 内层粒子（InnerParticle）填充 3D 爱心内部
- 粒子数量不变（1500 外层 + 600 内层）

## Task 2: 3D 投影 + 旋转 + 深度排序绘制

- 添加相机参数：`fov`（视野），`cameraZ`（相机距离）
- 添加旋转矩阵：绕 Y 轴旋转（自转）+ 绕 X/Z 轴旋转（拖拽）
- 透视投影公式：`screenX = x * fov / (fov + z) + centerX`
- 按 Z 值从远到近排序粒子，先画远的再画近的（深度排序）
- 粒子大小和透明度根据 Z 深度做微调（远的更小更淡）

## Task 3: 鼠标拖拽交互 + 心跳 3D 脉冲

- 监听 `mousedown/mousemove/mouseup` 和 `touchstart/touchmove/touchend`
- 拖拽时改变旋转角度，有惯性缓动
- 心跳时不仅散射，还叠加 Z 轴方向的脉冲缩放（爱心在 3D 空间"膨胀收缩"）
- 自转速度降低到更优雅的节奏

## Task 4: 保留现有效果 + 测试验证

- 保留：粒子闪烁（twinkle）、漂移（drift）、散射（scatter）、背景星星、颜色渐变
- 保留：心跳检测逻辑（lub-dub 双脉冲）
- 保留：resize 响应式
- 测试：桌面端拖拽旋转、移动端触摸旋转、心跳效果
